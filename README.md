# RoUnlock
RoUnlock is a free to use tool to remove the FPS lock on your roblox instance. 

# Steps to use
**NOTE** Run once after every update in order to keep your FPS unlocked forever. 

1. Double click main.py
2. Let it run
3. FPS Unlocked.
